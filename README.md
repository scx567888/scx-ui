## 开发注意事项

scx-ui 项目采用 Vite 进行开发<br>
开发环境 - 版本要求:

```
nodejs >= 14
```

运行

```
-> npm install
-> npm run dev
```

打包

```
-> npm run build
```

项目地址

```
https://github.com/scx567888/scx-ui
```

联系方式

```

```

注意事项 !!

```
1,关于 资源文件 图片 音效 css 第三方插件等 请放到 public/_assets 目录下.
```
